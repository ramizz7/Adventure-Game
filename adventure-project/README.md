# Adventure Game

## 👤 Auteur

Umar

## 📖 Beschrijving

Dit project is een tekst-gebaseerd spel waarin de speler keuzes maakt via de terminal.

De keuzes van de speler bepalen hoe het verhaal verder gaat en hoe het spel eindigt.

---

## ▶️ Hoe starten

```bash
node src/adventure.js
```

---

## 🛠️ Gebruikte technieken

* JavaScript (Node.js)
* readline (voor input)
* if/else (voor keuzes)
